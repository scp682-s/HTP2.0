### 片段2：拖拽阈值判断（行1503-1516）

**功能说明：** 计算手指移动距离，超过10px阈值才确认为拖拽操作，避免误触发拖拽

```javascript
onDragMove(e) {
    if (!this.dragSource || this.gameState !== 'playing') return;
    
    const touch = e.touches[0];
    const dx = touch.clientX - this.dragStartPos.x;
    const dy = touch.clientY - this.dragStartPos.y;
    const distance = Math.sqrt(dx * dx + dy * dy);
    
    // 超过阈值才开始拖动
    if (!this.isDragging && distance > this.dragThreshold) {
        this.isDragging = true;
        this.startDragVisual(touch);
        e.preventDefault();
    }
}
```
