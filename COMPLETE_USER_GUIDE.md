## 二、使用手册

### 2.1 产品简介

“房树人拼图游戏”是一款面向大学生心理自我觉察的创新应用。用户通过完成拼图任务，系统自动分析拼图行为和图像内容，生成个性化的心理分析报告，帮助用户更好地理解自身心理状态。

### 2.2 适用场景

- 高校心理健康教育活动
- 大学生心理普查与自我评估
- 心理课程教学演示
- 心理辅导体验活动
- 学生作品展示

### 2.3 使用流程

**第一步：启动系统**
- 启动后端服务程序
- 打开前端页面（可直接打开 index.html 或通过本地服务器访问）

**第二步：开始游戏**
- 点击”开始游戏”按钮
- 选择内置图像或上传自己的图片
  - 注意：上传图片必须同时包含”房子、树、人物”三个元素

**第三步：选择难度**
- 根据个人喜好选择拼图难度
- 系统自动生成拼图

**第四步：完成拼图**
- 通过拖拽、交换等操作完成拼图
- 系统记录所有操作行为

**第五步：查看报告**
- 拼图完成后，点击查看心理分析报告
- 报告包含多维度的心理状态分析和建议

### 2.4 报告内容说明

系统生成的心理分析报告包含以下部分：

- **总体状态观察**：基于拼图行为的整体评估
- **房屋解读**：从房屋图像分析安全感与归属感
- **树木解读**：从树木图像分析生命力与成长状态
- **人物解读**：从人物图像分析自我认知与人际关系
- **校园情境关联**：结合大学生活场景的针对性分析
- **可执行建议**：具体的心理调适建议
- **一句话心理提示**：简洁的核心提示

### 方式二：本地部署

如果您希望在本地环境运行系统，可按照以下步骤进行部署。

#### 3.2.1 环境要求

- **操作系统**：Windows / Mac / Linux
- **Python版本**：Python 3.8 或更高版本（推荐 Python 3.10/3.11）
- **网络要求**：需要联网访问AI服务

#### 3.2.2 快速安装

**Windows用户**

1. 双击运行项目根目录下的 install.bat 文件
2. 按照提示完成依赖安装
3. 打开 backend 文件夹，找到 .env.example 文件
4. 复制 .env.example 并重命名为 .env
5. 在 .env 文件中填入您的 DEEPSEEK_API_KEY（必填）
6. 双击运行启动脚本或手动启动后端服务

**Mac/Linux用户**

1. 打开终端，进入项目目录
2. 运行命令：`chmod +x install.sh`
3. 执行安装脚本：`./install.sh`
4. 进入 backend 文件夹：`cd backend`
5. 复制配置文件：`cp .env.example .env`
6. 编辑 .env 文件，填写 DEEPSEEK_API_KEY
7. 启动后端服务

#### 3.2.3 手动安装步骤

**步骤一：安装Python依赖**

1. 打开命令行工具（Windows使用CMD或PowerShell，Mac/Linux使用Terminal）
2. 进入项目的 backend 文件夹
3. 执行以下命令安装依赖：
   ```
   pip install -r requirements.txt
   ```
   或使用国内镜像加速：
   ```
   pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
   ```

**步骤二：配置环境变量**

1. 在 backend 文件夹中找到 .env.example 文件
2. 复制该文件并重命名为 .env
   - Windows: `copy .env.example .env`
   - Mac/Linux: `cp .env.example .env`
3. 使用文本编辑器打开 .env 文件
4. 填入必要的配置信息：
   ```
   DEEPSEEK_API_KEY=sk-xxxxxxxxxxxxxxxx（必填）
   BAILIAN_API_KEY=sk-xxxxxxxxxxxxxxxx（可选）
   CUSTOM_IMAGE_MAX_MB=20（可选，默认20）
   BAILIAN_IMAGE_MAX_MB=9（可选，默认9）
   ```

**步骤三：启动后端服务**

1. 在 backend 文件夹中执行：
   ```
   python app.py
   ```
2. 看到类似以下提示表示启动成功：
   ```
   * Running on http://127.0.0.1:5000
   * Running on http://0.0.0.0:5000
   ```
3. 后端服务默认监听 5000 端口

**步骤四：访问前端页面**

有两种方式访问前端：

**方式A：直接打开HTML文件**
- 找到项目根目录下的 index.html 文件
- 双击打开即可在浏览器中使用

**方式B：使用本地服务器（推荐）**
1. 打开新的命令行窗口
2. 进入项目根目录
3. 执行以下命令启动静态服务器：
   ```
   python -m http.server 8000
   ```
4. 在浏览器中访问：`http://localhost:8000`

#### 3.2.4 验证安装

安装完成后，可通过以下步骤验证：

1. 访问前端页面，检查是否正常显示
2. 点击"开始游戏"按钮
3. 选择一张内置图片
4. 选择拼图难度并生成拼图
5. 完成拼图后查看是否能正常生成心理分析报告

如果以上步骤都能正常完成，说明安装成功。

#### 3.2.5 获取API密钥

**DeepSeek API密钥获取方法：**

1. 访问 DeepSeek 官网：https://platform.deepseek.com/
2. 注册并登录账号
3. 进入API管理页面
4. 创建新的API密钥
5. 复制密钥并填入 .env 文件

**其他AI服务：**

如需使用其他AI服务（通义千问、智谱AI、百度文心一言等），请参考 backend/API_EXAMPLES.md 文档中的详细说明。

## 六、常见问题与注意事项

### 6.1 使用注意事项

**图片要求**
- 系统提供4张预设图片，可直接使用
- 上传自定义图片时，必须同时包含”房子、树、人物”三个元素
- 图片大小建议不超过10MB，确保上传和分析速度
- 推荐使用清晰、完整的图片，以获得更准确的分析结果

**操作建议**
- 首次使用建议选择系统内置图片
- 拼图过程中请保持网络连接稳定
- 报告生成需要一定时间，请耐心等待
- 建议在安静的环境中完成拼图，以获得更真实的行为数据

### 6.2 常见问题解答

**问：为什么我上传的图片无法使用？**
答：请确认图片中同时包含”房子、树、人物”三个元素。系统会自动检测图片内容，只有包含这三个元素的图片才能进行心理分析。

**问：拼图难度如何选择？**
答：建议根据个人情况选择：
- 简单模式：适合首次体验或时间有限的用户
- 中等模式：适合大多数用户，体验较为完整
- 困难模式：适合喜欢挑战的用户

**问：报告生成需要多长时间？**
答：通常需要10-30秒，具体时间取决于网络状况和服务器负载。

**问：可以重复测试吗？**
答：可以。每次测试都会生成新的报告，系统会记录您的行为变化趋势。

**问：报告结果准确吗？**
答：报告基于心理学”房树人测验”理论和您的拼图行为数据生成，仅供参考和自我觉察使用，不能作为医学诊断依据。

### 6.3 技术问题排查

**页面无法打开**
- 检查网络连接是否正常
- 确认访问地址是否正确：http://119.23.144.133:5251/
- 尝试刷新页面或更换浏览器

**图片上传失败**
- 检查图片大小是否超出限制
- 确认图片格式（支持JPG、PNG等常见格式）
- 确认图片内容是否包含三要素

**拼图无法操作**
- 检查是否已选择图片并生成拼图
- 尝试刷新页面重新开始
- 确认浏览器是否支持（建议使用现代浏览器）

**报告无法生成**
- 确认已完成拼图
- 检查网络连接是否稳定
- 如长时间无响应，可刷新页面重试
