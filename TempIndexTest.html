checkPieceCorrect() 函数 - 碎片位置正确性判断

被测函数代码

function checkPieceCorrect(piece, cell) {
    // 条件A：碎片是否存在
    if (!piece) return false;
    
    // 获取坐标
    const cellRow = parseInt(cell.dataset.row);
    const cellCol = parseInt(cell.dataset.col);
    const originalRow = parseInt(piece.dataset.originalRow);
    const originalCol = parseInt(piece.dataset.originalCol);
    
    // 条件D：碎片是否被旋转
    const isRotated = piece.dataset.rotated === 'true';
    
    // 条件B&C&!D：行匹配 且 列匹配 且 未旋转
    if (originalRow === cellRow && originalCol === cellCol && !isRotated) {
        return true;
    } else {
        return false;
    }
}
判断条件：
A: 碎片存在 | B: 行匹配 | C: 列匹配 | D: 已旋转
返回 true 条件： A && B && C && !D
